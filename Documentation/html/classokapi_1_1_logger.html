A logger that opens the input file by name. If the file contains /ser/, the file will be opened in write mode. Otherwise, the file will be opened in append mode. The file will be closed when the logger is destructed.

Parameters
itimerA timer used to get the current time for log statements.
ifileNameThe name of the log file to open.
ilevelThe log level. Log statements more verbose than this level will be disabled.
