\hypertarget{index_intro_sec}{}\doxysection{Introduction}\label{index_intro_sec}
{\bfseries{Okapi\+Lib}} is a PROS library for programming VEX V5 robots. This library is intended to raise the floor for teams with all levels of experience. New teams should have an easier time getting their robot up and running, and veteran teams should find that Okapi\+Lib doesn\textquotesingle{}t get in the way or place any limits on functionality.

For tutorials on how to get the most out of Okapi\+Lib, see the Tutorials section. For documentation on using the Okapi\+Lib API, see the API section.\hypertarget{index_getting_started}{}\doxysection{Getting Started}\label{index_getting_started}
Not sure where to start? Take a look at the Getting Started tutorial. Once you have Okapi\+Lib set up, check out the Clawbot tutorial.\hypertarget{index_using_docs}{}\doxysection{Using The Documentation}\label{index_using_docs}
